# Credit Risk Model — Give Me Some Credit

Modelo de clasificación binaria para predecir incumplimiento de pago en los próximos 
dos años, usando el dataset [Give Me Some Credit](https://www.kaggle.com/c/GiveMeSomeCredit) de Kaggle.

## Resultados
| Modelo | AUC-ROC (CV) |
|---|---|
| Regresión Logística | 0.8550 ± 0.0043 |
| Árbol de Decisión | 0.8469 ± 0.0043 |
| Random Forest | 0.8615 ± 0.0039 |
